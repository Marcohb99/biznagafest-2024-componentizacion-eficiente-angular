# Backend

## Instalación

```bash
pnpm install
```

## Ejecución

```bash
pnpm start
```

## API

### GET /todos

Devuelve todas las tareas.

### POST /todos

Crea una nueva tarea.

### DELETE /todos/:id

Elimina una tarea por su ID.

### PUT /todos/:id

Actualiza una tarea por su ID.
